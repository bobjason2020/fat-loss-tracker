# 更新日志

本项目的所有重要更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，

## [1.0.0] - 2026-03-11

### 新增
- 配置化系统，支持自定义记录存放路径
- 初始化引导流程，首次使用自动引导配置
- 睡眠相关触发关键词支持
- `config.yaml` 配置文件
- `config.example.yaml` 配置示例
- MIT 开源许可证
- README.md 项目文档
- CHANGELOG.md 更新日志
- .gitignore Git 忽略规则

### 优化
- 移除所有硬编码路径，改为从配置读取
- 触发条件新增睡眠相关关键词
- 完善文档和使用示例

### 修复
- 无
